# Legal Pages Checklist — NutriFit V1

Estado: mínimo requerido para lanzamiento comercial inicial.
Última actualización: 2026-03-30.

## 1) Privacy Policy (página pública)

- [ ] URL pública estable y accesible sin login.
- [ ] Responsable del tratamiento y correo de contacto.
- [ ] Qué datos se recolectan (auth, uso app, datos wellness, billing metadata).
- [ ] Para qué se usan los datos (operación app, mejora producto, soporte).
- [ ] Integraciones de terceros declaradas (Firebase, Google Play Billing, OpenAI server-side).
- [ ] Retención y eliminación de datos explicada.
- [ ] Derechos del usuario y cómo ejercerlos.
- [ ] Fecha de última actualización y política de cambios.

## 2) Support page (página pública)

- [ ] Canal de soporte (email/formulario) operativo.
- [ ] Tiempo estimado de respuesta.
- [ ] FAQ mínimo (login, suscripción, reembolsos, restauración, problemas IA).
- [ ] Instrucciones para reportar bugs con versión/dispositivo.

## 3) Delete account page (página pública)

- [ ] Flujo claro para solicitar eliminación de cuenta.
- [ ] Método de verificación de identidad del solicitante.
- [ ] Alcance de borrado: cuenta, datos de perfil, datos de uso.
- [ ] Excepciones legales/financieras (si aplica) claramente descritas.
- [ ] SLA estimado para completar eliminación.
- [ ] Confirmación al usuario cuando el borrado finaliza.

## 4) Alineación con Play Console

- [ ] URLs de Privacy Policy, Support y Delete Account cargadas en Play Console.
- [ ] Texto legal consistente entre app, web y Data Safety.
- [ ] Evidencia interna (capturas/links) guardada para auditoría.

## 5) Boundary de seguridad de producto

- [ ] Disclaimer explícito: NutriFit es guía de bienestar, no diagnóstico/tratamiento médico.
- [ ] Recomendaciones IA presentadas como orientación general, no clínica.
