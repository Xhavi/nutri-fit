# Roadmap — NutriFit

## Phase 0 — Foundation (current)
- Define product vision, scope, and architecture baseline
- Establish repository agreements and workflow conventions
- Prepare implementation plan for Flutter bootstrap

## Phase 1 — Project bootstrap
- Initialize Flutter app scaffold
- Configure environments and base app structure
- Implement navigation shell and design system primitives

## Phase 2 — Core tracking MVP
- Meal logging flows
- Exercise logging flows
- Progress dashboard baseline
- Local validation and analytics event baseline

## Phase 3 — AI coach MVP
- Chat interface and session handling
- Serverless AI orchestration backend
- Safety and non-medical guardrails
- Basic personalization context

## Phase 4 — Stabilization and beta
- QA hardening, performance tuning, and crash monitoring
- Closed beta feedback loop
- Iterative UX improvements

## Phase 5 — Expansion
- Health Connect / HealthKit integration
- Advanced coaching plans and insights
- Growth features and monetization experiments
